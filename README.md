# 🎬 IMDB Analysis Project

## 📌 Project Description

This project analyzes the IMDB Top 1000 movies dataset using Python and Pandas.

The goal is to explore movie trends, ratings, genres, and other insights through data analysis and visualization.

---

## ⚙️ Technologies

- Python  
- Pandas  
- Matplotlib  

---

## ✨ Features

- 🎭 Genre-based analysis  
- 📅 Decade analysis  
- 🏆 Top-rated movies  
- 📊 Data visualization  

---

## 📊 Analysis Includes

- Number of movies per genre  
- Average ratings by genre  
- Movies grouped by decades  
- Highest-rated and most popular movies  

---

## 🚀 Note

This project was created for educational purposes and serves as a practice for data analysis using Python.
